# Zomato Data Analysis using Python

## Project Overview
This project performs exploratory data analysis (EDA) on a Zomato restaurant dataset using Python. The analysis helps in understanding restaurant trends, customer preferences, ratings, voting patterns, online ordering behavior, and restaurant categories through data visualization.

The project uses data preprocessing techniques and visualization libraries to analyze restaurant-related information and generate meaningful insights from the dataset.

## Features
- Import and preprocess restaurant dataset from CSV file
- Clean restaurant ratings data
- Perform data inspection and null-value checking
- Analyze restaurant categories and voting trends
- Identify restaurants with the highest votes
- Study online ordering behavior
- Visualize restaurant rating distribution
- Analyze approximate cost for two people
- Generate a heatmap for restaurant type vs online ordering availability

## Technologies Used
- Python
- Pandas
- NumPy
- Matplotlib
- Seaborn

## Dataset
The dataset used in this project is:

`Zomato-data-.csv`

It contains restaurant-related information such as:
- Restaurant name
- Ratings
- Votes
- Restaurant type
- Online order availability
- Approximate cost for two people
